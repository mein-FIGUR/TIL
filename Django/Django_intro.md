# Django 프로젝트 생성 과정

- 프로젝트에 가상환경 생성

```
python -m venv venv
```



- 가상 환경 활성화

```
source venv/Scripts/activate
```



- 모듈 확인

```
pip list
```



- django 모듈 설치

```
pip install django
```



- django 프로젝트 초기화

```
django-admin startproject django_intro .
```



- application 생성

```
python manage.py startapp <application이름>
```



- 서버 실행

```
python manage.py runserver
```
