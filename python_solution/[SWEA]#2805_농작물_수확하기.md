#### 내가 쓴 풀이(성공)

```python
T = int(input())

for test_case in range(1, T+1):
    size = int(input())
    mid = size// 2
    result = 0
    
    for i in range(size):
        now = input()
        farm_line = list(map(int,list(now)))
        if i <= mid :
            result += sum(farm_line[mid-i:mid+i+1])
        else:
            result += sum(farm_line[mid-(size-i-1):mid+(size-i-1)+1])
    print(f'#{test_case} {result}')
```
